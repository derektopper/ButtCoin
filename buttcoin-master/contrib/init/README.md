Sample configuration files for:
```
SystemD: buttcoind.service
Upstart: buttcoind.conf
OpenRC:  buttcoind.openrc
         buttcoind.openrcconf
CentOS:  buttcoind.init
OS X:    org.buttcoin.buttcoind.plist
```
